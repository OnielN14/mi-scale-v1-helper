# Sample
Xiaomi Mi Scale V1 Web Bluetooth API sample implementation 

## Setup, Build and Serve
```sh
# Install Dependecies
yarn install

# serve with hot reload at localhost:1234
yarn run dev

# build for production
yarn run build
```
